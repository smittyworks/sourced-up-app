# SourcedUp -- Find Independent Journalists

SourcedUp is a unified feed for independent journalism — one place to follow your favorite independent journalists across Substack, YouTube, Rumble, podcasts and more.

  🔗 [Sourced Up](https://www.sourcedup.news/)

  🔗 [Directory of independent journalists](https://www.sourcedup.news/directory)
  
  Independent journalism is booming, but keeping up is fragmented. SourcedUp solves that by aggregating content from across platforms into a single chronological feed, with deep links back to the original source.

  Key features (roadmap):

  - Curated journalist directory (Live) – Browse and follow vetted independent journalists
  - Unified feed – All content in one place, regardless of platform
  - Real-time notifications – Get alerted when followed journalists publish
  - Deep linking – Tap through directly to Substack, YouTube, Rumble, etc.
  - Platform filtering – Filter by format or source
